Grab DocuClerk.exe using the button above. It is a single file, so there is nothing to install.

3

Analyze

Drop in a contract and get your first safety score in seconds. No sign-up required.

First launch on Windows

DocuClerk is a new app and is not code-signed yet, so Windows may show a "Windows protected your PC" screen the first time you open it. That is SmartScreen being careful about a publisher it has not seen before, not a problem with the file. Click More info, then Run anyway, and you will only need to do it once.
